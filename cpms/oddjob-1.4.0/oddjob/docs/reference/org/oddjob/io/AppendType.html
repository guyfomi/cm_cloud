append

Specify a file for appending to.

Property Summary

file The file path.

Example Summary

Example 1 Append a line to a file.

Property Detail

file
Configured ByATTRIBUTE
AccessREAD_WRITE
RequiredYes.

The file path.

Examples

Example 1

Append a line to a file. 

<oddjob id="this">
    <job>
        <sequential>
            <jobs>
                <variables id="vars">
                    <myFile>
                        <append file="${this.args[0]}/messages.txt"/>
                    </myFile>
                </variables>
                <echo>
                    <output>
                        <value value="${vars.myFile}"/>
                    </output>Hello World</echo>
                <echo>
                    <output>
                        <value value="${vars.myFile}"/>
                    </output>Goodbye World</echo>
            </jobs>
        </sequential>
    </job>
</oddjob>
