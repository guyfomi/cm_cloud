bean-bus

A job that allows the construction of a org.oddjob.beanbus.BeanBus.

A Bean Bus is an assembly of java.util.Collections.

name A name, can be any text.

Examples

Example 1

A simple bus example. 

<oddjob>
    <job>
        <bean class="org.oddjob.beanbus.mega.MegaBeanBus" id="bus">
            <parts>
                <bus:iterable-driver xmlns:bus="oddjob:beanbus">
                    <beans>
                        <list>
                            <values>
                                <value value="Apple"/>
                                <value value="Orange"/>
                                <value value="Pear"/>
                            </values>
                        </list>
                    </beans>
                </bus:iterable-driver>
                <bus:bean-capture id="list" xmlns:bus="oddjob:beanbus"/>
            </parts>
        </bean>
    </job>
</oddjob>
