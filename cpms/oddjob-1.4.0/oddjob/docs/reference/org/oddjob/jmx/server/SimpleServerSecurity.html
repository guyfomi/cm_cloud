jmx:server-security

Provide a JMX simple security environment for a jmx:server.

If SSL is used the appropriate JVM parameters need to be set for both client and server. See The JMX Tutorial.

Property Summary

accessFile The location of the access file.
passwordFile The location of the password file.
useSSL Use Secure Sockets (SSL).
